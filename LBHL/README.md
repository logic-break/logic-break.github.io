# LBHL (Logic-Break Half-Life)

Короче, это Халфа в браузере. Портанул, чтобы работало прямо из-под капота веба. 

### 🛠 Че по чем:
* **Движок:** Юзает WebAssembly и WebGL, так что графон отрисовывается видяхой через браузер.
* **База:** Взял исходники у [Pixelsuft/hl](https://github.com/Pixelsuft/hl), допилил напильником, чтобы всё это взлетело как надо.
* **Зачем:** Делал для себя (Logic-Break), чисто по фану, но решил выложить — пользуйтесь.
